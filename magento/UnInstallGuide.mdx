## A. Uninstall SureBright module

<Accordion title=" Turn Off SureBright Integration">
  Run this command to disable the module and remove old files:

  > php bin/magento module:disable Surebright_Integration --clear-static-content
</Accordion>

## B. Re-Install Steps

<Accordion title="Re-Install SureBright" description="Skip this step if you only disbaled the mdule and haven't removed it yet.">
  Run this to add SureBright back:

  > composer require surebright/integration
</Accordion>

**If you have only disbaled the app then just follow below step and ignore rest of guide.**

<Accordion title=" Turn SureBright Back On">
  Run this to enable the module:

  > php bin/magento module:enable Surebright_Integration
</Accordion>

**Follow below guide only if you had removed the module.**

<Accordion title=" Update Magento and Rebuild Code" iconType="duotone">
  Run this to finalize the reinstall:

  > bin/magento setup:upgrade
  >
  > bin/magento setup:di:compile
</Accordion>

<Accordion title=" Clear Cache Again" iconType="duotone">
  Run this to finish the reinstall:

  > bin/magento cache:clean
</Accordion>
