# HTTP Status Codes & Error Types

| Code | Message |
|------|---------|
| 511 | Insufficient Parameters |
| 512 | Validation Failed for Input Parameters |
| 513 | Product Not Found |
| 514 | No Quote Found |
| 515 | Store Not Found |
| 516 | Not Eligible |
| 517 | Category Not Found |
| 200 | Warranty Created Successfully |
| 500 | Something Unexpectedly Went Wrong |
