## Headers

| Header | Description | Required |
|--------|-------------|----------|
| X-SureBright-Access-Token | Access token for authentication | No |





# Examples

## Request

```bash
curl -X GET 'https://{HOST_NAME}/quoteService/api/v1/store/listCategories/{storeId}?country=US&state=AL'
```
